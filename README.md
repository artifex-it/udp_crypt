# udp_crypt

Server e client per l'invio di datagrammi UDP cifrati.

Gli algoritmi disponibili sono:
- cifrario di Cesare
- ROT13
- adbash
- cifrario carbonaro
- cifrario affine
- cifrario di Vigenère
- cifrario baconiano
